# Kit de backlinks — Hacé Cuentas

Material listo para construir la autoridad de dominio de [hacecuentas.com](https://hacecuentas.com) de forma white-hat. Todo acá está **listo para copiar y pegar** o ejecutar con mínima intervención manual.

## Qué esperar de resultado

- **Semana 1**: 10-15 backlinks DR 60+ activos (directorios + posts auto-publicados)
- **Semana 2**: 5-10 backlinks DR 80+ adicionales (Medium, Dev.to indexan)
- **Semana 4**: primeros inbound links orgánicos de personas que encontraron los posts
- **Mes 2-3**: impacto en rankings de calcs secundarias (long-tail)
- **Mes 6**: tráfico referral agregando +10-20% al orgánico

## Reglas críticas

1. **No hacer todo el mismo día** — distribuir en 2-3 semanas. Un pico brusco de backlinks se ve artificial.
2. **Personalizar cada submit** — los directorios con copy genérico se rechazan o pierden valor.
3. **Nunca comprar links** — ni uno. La penalización destruye meses de trabajo.
4. **Responder cada comentario** en los posts publicados — señal de sitio activo.
5. **Guardar credenciales** de cada plataforma en el password manager para updates futuros.
